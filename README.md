# Python Quiz Game

This is a simple quiz game implemented in Python using the Tkinter GUI framework.

## Installation
- Clone the repository:
git clone https://github.com/TristynWS/Python-Quiz-Game.git

## Usage
- Run the quiz game:
python self_quiz_game.py
